# PerfumeriaJava

Sistema de gestión de perfumería desarrollado en Java, NetBeans y MySQL.

## Descripción

Este proyecto consiste en una aplicación de escritorio para la administración de una perfumería. Permite gestionar clientes, proveedores, perfumes, inventario y ventas mediante una interfaz gráfica desarrollada en Java.

El proyecto implementa los patrones de diseño **MVC (Modelo-Vista-Controlador)** y **DAO (Data Access Object)** para mantener una estructura organizada y facilitar el mantenimiento del código.

---

## Tecnologías utilizadas

- Java
- NetBeans
- MySQL
- JDBC
- MVC
- DAO

---

## Funcionalidades

- Inicio de sesión.
- Gestión de clientes.
- Gestión de perfumes.
- Gestión de proveedores.
- Control de inventario.
- Registro de ventas.
- Conexión con base de datos MySQL.

---

## Estructura del proyecto

```
src/
├── conexion/
├── controlador/
├── dao/
├── modelo/
├── vista/
└── main/
```

---

## Cómo ejecutar el proyecto

1. Abrir el proyecto en NetBeans.
2. Crear la base de datos en MySQL.
3. Configurar la conexión en la clase `Conexion.java`.
4. Ejecutar la clase `Main.java`.
